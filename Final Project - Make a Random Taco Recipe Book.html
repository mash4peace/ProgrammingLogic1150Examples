For this project, you'll use this API

  • Random Taco API  http://taco-randomizer.herokuapp.com/random/?full_taco=true 

and these libraries

  • requests
  • python-docx
  • pillow

To create a random taco recipe book. Use Python for all of these tasks.

Your first program should: (25 points)

Search Unsplash for a taco image. (for example https://unsplash.com/photos/JiRSy0GfqPA) Save the image on your computer.

  • The downloaded image is very large. Use pillow resize the image to a smaller size, perhaps 800px wide. Make sure you preserve the aspect ratio.
  • Write the text "Random Taco Cookbook" on the image.
  • Save to a file. 

Your second program should: (75 points)

  • Use requests to download three random tacos from the random taco API. Save the data for each of three tacos in your program. Notice that each recipe is divided into five sections for base_layer, seasoning, mixin, condiment, and shell.
  • Use Python to create a Word document. 
    • On the first page, insert the text "Random Taco Cookbook"
    • On the first page, add the resized taco image
    • On the first page, write the name of the image author
    • On the first page, write a link to the random taco API
    • On the first page, write your own name.
    • On the second page, write the first taco recipe. Use a larger font for the heading for each of the sections.
    • On the third page, repeat for the next recipe.
    • On the fourth page, repeat for the third recipe.
  • Save your word document.

To upload: your Python code, and an example word document created with your program. 

Extra credit + 15 points: register for the Unsplash API https://unsplash.com/developers and use this to download an image for your book.  You'll need to read documentation and figure out how to use requests to search, and then how to download an image.  If you do this, you'll only need to write one program to create the entire recipe book with image, and text.
